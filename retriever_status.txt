Notes

API Retriever is code-real with a guarded allowlisted HTTP connector. It reads configured endpoints from RAG_API_RETRIEVER_ENDPOINTS, restricts calls by RAG_API_ALLOWED_HOSTS, normalizes JSON/text responses into evidence rows, and scores API results without inventing live data when no endpoint is configured. Graph Retriever is code-real with local entity-edge graph seed nodes, multi-hop graph expansion, edge evidence, chunk-distance scoring, and graph-neighbor candidate expansion. Agentic Retriever is code-real with optional CrewAI planning and deterministic fallback. Knowledge Graph Retriever is code-real with a local entity-relation graph over chunks, sections, documents, standards, and references. Late Interaction Retriever is code-real with token-level max-sim scoring for exact terms, identifiers, numeric values, stems, fuzzy matches, and character n-grams. Memory Retriever is code-real with chat-history source memory, follow-up expansion, previous source IDs, sections, entities, and standards. SQL / Database Retriever is code-real with safe parameterized SQLite retrieval over documents, chunks, metadata, retrieval logs, and chat history. Multi-hop Retriever is code-real with staged hop planning, hop query expansion, per-hop scoring, linked chunk expansion, section sibling expansion, and relationship evidence. Iterative Retriever is code-real with first-pass analysis, missing-term detection, focused retry queries, candidate merging, and iterative rescoring. Query-decomposition Retriever is code-real with typed subquestion parsing, query expansion, and per-subquestion coverage scoring. Tool-aware Retriever is code-real with a local tool planner that selects vector, keyword, table, SQL, graph, memory, iterative, multimodal, citation, API, and late-interaction retrieval behavior, then expands queries and rescoring candidates. Ontology Retriever is code-real with a local engineering ontology, concept detection, synonym/broader/related-term expansion, metadata matching, and candidate rescoring. Symbolic Retriever is code-real with a local rule engine for obligations, prohibitions, conditions, exceptions, permissions, recommendations, definitions, numeric constraints, and safety-critical wording. Semantic Graph Retriever is code-real with dense/vector relevance fused with knowledge-graph nodes, ontology concepts, relationship types, section links, and lexical query terms.
